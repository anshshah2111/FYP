Landmark Detection Module

This module is employed in the training phase. Matterport Simulator generates a navigation graph from the 3D scan. In the graph, each node represents a navigable viewpoint.

YOLOv3 algorithm is selected, will apply a single neural network to the full image. See Working.

We maintain a list of the percentage area of bounding box at each node. We choose the node that has the largest percentage of that particular landmark.

The Matterport 3D Dataset

The Matterport 3D dataset, which is the largest currently available RGB-D dataset developed by the research team of Australian Centre for Robotic Vision.
It has the most extensive depth image collection which includes multiple navigable points and allows multiple trajectories for simulating motion. Each navigable viewpoint has a panoramic image which captures the entire sphere of the visible scene from that viewpoint
