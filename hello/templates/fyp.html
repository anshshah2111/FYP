Landmark Detection Module

This module is employed in the training phase. Matterport Simulator generates a navigation graph from the 3D scan. In the graph, each node represents a navigable viewpoint.

YOLOv3 algorithm is selected, will apply a single neural network to the full image. See Working.

We maintain a list of the percentage area of bounding box at each node. We choose the node that has the largest percentage of that particular landmark.

Natural Language Processing

This module converts your speech instruction to text using Google's proprietary speech-text API and then passes this instruction to a LSTM that performs Named Entity Recognition and extracts the intended goal location to pass it on to the path determination module

See demo

Path Determination

Using the Speech Processing algorithm we extract the keyword which represents goal location. In the final module, we generate the trajectory to be followed to reach the goal. For this purpose we use the information generated in the training phase.

Finally once we have the goal node, we find the optimum path to traverse through the map. The optimum path here is the shortest path from the initial node to the goal node.

Dijsktra’s algorithm would perform the best with a worst case time complexity of O of E + V log V where E is the number of edges and V is the number of vertices in the graph.

Alternative Algorithms

Shortest Path: Dijkstra's Algorithm

Dijkstra's algorithm has many variants but the most common one is to find the shortest paths from the source vertex to all other vertices in the graph generated for Navigation

Explore

Shortest Path: Floyd-Warshall's Algorithm

It is used to find the shortest paths between between all pairs of vertices in a graph, where each edge in the graph has a weight which is positive or negative.

Explore

Shortest Path: Bellman Ford's Algorithm

It depends on the relaxation principle where the shortest distance for all vertices is gradually replaced by more accurate values until it reaches the optimum solution

Explore

Eccentricity of a Goal Location

The distance from a particular location to all other nodes is taken and among those distances, it is the highest of distances. The same can be calculated for every node in the path.

Explore

Flexible Initial Location of Agent

While most navigation tasks happen under a precondition of a fixed location, traversal in the indoor environment can be further carried out with an option of changing the initial location

Description of Navigable Path

An extensive description of the navigation to the goal location in isolation; details of no. of nodes traversed, distances between them and complete traversal logs

Breadth First Search

Graphs may contain cycles, so to avoid processing a node more than once, we use a boolean visited array & assumed that all nodes are reachable from the starting node

The Matterport 3D Dataset

The Matterport 3D dataset, which is the largest currently available RGB-D dataset developed by the research team of Australian Centre for Robotic Vision.
It has the most extensive depth image collection which includes multiple navigable points and allows multiple trajectories for simulating motion. Each navigable viewpoint has a panoramic image which captures the entire sphere of the visible scene from that viewpoint
